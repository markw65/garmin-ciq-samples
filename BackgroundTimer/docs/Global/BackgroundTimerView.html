Instance Method Details

backgroundEvent() as Void

If we do receive a background event while the application is open, go ahead and reset to the default timer.

deleteBackgroundEvent() as Void

Delete the background event. We can get rid of this event when the application opens because now we can see exactly when the timer is going to expire. We will set it again when the application closes.

    Device Context

requestUpdate() as Void

This is the callback method we use for our timer. It is only needed to request display updates as the timer counts down so we see the updated time on the display.

resetTimer() as Lang.Boolean

If the timer is paused, then go ahead and reset it back to the default time.

Returns:

  • true if timer is reset, false otherwise

saveProperties() as Void

Save all the persisted values into the object store. This gets called by the Application base before the application shuts down.

setBackgroundEvent() as Void

Set up a background event to occur when the timer expires. This will alert the user that the timer has expired even if the application does not remain open.

startStopTimer() as Void

If the timer is running, pause it. Otherwise, start it up.
